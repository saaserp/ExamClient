Dependencies

The following is a list of dependencies for this project. These dependencies are required to compile and run the application:

Artifact ID Type Version URL Comment
commons-beanutils-core jar 1.7.0 http://jakarta.apache.org/commons/beanutils/
commons-digester jar 1.7 http://jakarta.apache.org/commons/digester/
commons-logging jar 1.0.4 http://jakarta.apache.org/commons/logging/
junit jar 3.8.1 http://www.junit.org Used only for unit testing
maven-cobertura-plugin plugin 1.1.1 http://maven-plugins.sourceforge.net/maven-cobertura-plugin/ Site Only Generates test coverage reports.
maven-artifact-plugin plugin 1.5 http://www.ibiblio.org/maven/maven/plugins/ Build Only Generates distributions.
maven-xdoc-plugin plugin 1.9.2 http://maven.apache.org/reference/plugins/xdoc/ Site Only - v1.9.2 (minimum) required for building the Site documentation.
xerces jar 2.4.0 http://xerces.apache.org/xerces2-j/index.html Any JAXP compliant parser (1.2 or later).
xml-apis jar 1.0.b2 http://xml.apache.org/commons/ Any JAXP compliant parser (1.2 or later)
